## 关于我

- 专注于**大语言模型**方向 — 预训练、微调、对齐与部署
- 从零搭建了完整的 LLM 训练流水线（Tokenizer -> 预训练 -> SFT -> 对话）
- 擅长 **LoRA/QLoRA 微调**、**CoT 数据工程**、**NPU 部署**
- 当前在深入研究 **RAG 系统**、**Multi-Agent 框架** 和 **GRPO 对齐**

> **my-llm** — 在 RTX 4060 上从零训练 29M 参数 Llama 风格语言模型
> - Byte-level BPE 分词器（8192 词表）
> - 预训练：loss 9.1 -> 2.3 | SFT：loss 3.3 -> 1.9
> - RoPE + GQA + SwiGLU + RMSNorm
